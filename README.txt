SITE EXPERTISE CONTABILIDADE

Arquivos incluídos:
- index.html
- style.css
- script.js

Como publicar:
1. Acesse o painel do servidor ou Plesk do domínio expertise.srv.br.
2. Abra o Gerenciador de Arquivos.
3. Entre na pasta raiz do site, normalmente chamada httpdocs, public_html ou wwwroot.
4. Envie os três arquivos desta pasta.
5. Abra https://expertise.srv.br no navegador.

Ajustes importantes antes de publicar:
- Confirmar o telefone/WhatsApp correto.
- Confirmar o e-mail novo. Usei contato@expertise.srv.br como sugestão.
- Confirmar se o link de webmail será o mesmo ou outro.
- Se desejar formulário funcionando sem abrir programa de e-mail, será necessário configurar envio SMTP no servidor.

Sugestão de DNS:
- expertise.srv.br deve apontar para o servidor onde os arquivos serão hospedados.
- Depois que o site novo estiver aprovado, o domínio antigo paeseandrade.srv.br pode redirecionar para expertise.srv.br.
